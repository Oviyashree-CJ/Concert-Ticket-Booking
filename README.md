# Concert Ticket Booking System

A full-stack web application that allows users to browse concerts, select seats, make bookings, and complete payments online.  
Built with **React (frontend)**, **Spring Boot (backend)**, and **MySQL (database)**.

---

## Tech Stack

### **Frontend**
- React.js (React Router, Axios, Framer Motion)
- Custom CSS for styling

### **Backend**
- Spring Boot (Java 17+)
- Spring Data JPA
- MySQL Database
- REST API with full CRUD endpoints

---

## Features

- **Concert Listings** — View available concerts with details.  
- **Seat Booking** — Interactive seat map with dynamic pricing (Platinum / Gold / Silver).  
- **Payment Simulation** — Card / UPI / Wallet options.  
- **Ticket Generation** — View and download your confirmed ticket.  
- **User Tracking** — Store user details and booking history.  
- **Admin Panel** — View all bookings and add concert details.
